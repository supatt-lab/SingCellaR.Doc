Contaminated cell-type removal

The cells in the cluster10 express genes such as ‘GZMB’ and ‘EPHA2’ suggesting a contaminating population of plasmacytic dendritic cells.

plot_umap_label_by_genes(human_HSPCs,gene_list = c("GZMB","EPHA2"))

Next, Supervised harmony integration will be performed. Howerver, it requires marker gene information identified for each individual cluster for the integration. This information is stored in the ‘marker.genes’ slot and it is overwritten by the previous ‘findMarkerGenes’ analysis.

To solve this problem, the ‘marker.genes’ information will be retrieved from the original SingCellaR integrative object and will be added in the existing ‘human_HSPCs’ object.

temp_human_HSPCs<-local(get(load(file="../SingCellaR_example_datasets/Psaila_et_al/SingCellaR_objects/human_HSPCs_v0.1.SingCellaR.rdata")))
human_HSPCs@marker.genes<-temp_human_HSPCs@marker.genes
rm(temp_human_HSPCs)

AUCell analysis

SingCellaR incorporates AUCell analysis for the AUCell score calculation. To run the analysis, AUCell package needs to be installed. See more details at https://www.bioconductor.org/packages/release/bioc/html/AUCell.html.

First, the function ‘AUCell_buildRankings’ will be performed and the AUCell_buildRankings object will be saved.

library(AUCell)
## 
## Attaching package: 'AUCell'
## The following object is masked from 'package:base':
## 
##     cbind
set.seed(1)
exprMatrix<-get_umi_count(human_HSPCs)
human_HSPCs_cells_rankings <- AUCell_buildRankings(exprMatrix, nCores=4, plotStats=TRUE)
## Quantiles for the number of genes detected by cell: 
## (Non-detected genes are shuffled at the end of the ranking. Keep it in mind when choosing the threshold for calculating the AUC).
##     min      1%      5%     10%     50%    100% 
##  517.00  842.99 1233.90 1436.90 2709.00 7947.00
## Using 4 cores.

save(human_HSPCs_cells_rankings,file="../SingCellaR_example_datasets/Psaila_et_al/SingCellaR_objects/human_life_cells_rankings.AUCells.rdata")
